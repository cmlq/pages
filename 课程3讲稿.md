# 课程3讲稿

## [封面]

这一讲会围绕“从页面结构到交互开发”这条主线展开，先帮助大家建立基础认知，再逐步进入 HTML、CSS 和 JavaScript 的学习。

这门课的目标不是先背概念，而是先理解前端、后端、数据库和 API 是怎么协作的，再把前端基础技术串成一条完整的学习路径。

## [第一页]
这一页是整门课的目录，一共分成两大部分。

第一部分是“基础学习路径”，对应基础认知、前端、后端和数据库四个环节。这里会先讲清楚 Web 应用的基本链路，让大家明白前端为什么要和后端、数据库一起配合。

第二部分是“前端基础技术（基础入门）”，核心内容是 HTML、CSS 和 JS。我们会按“结构、样式、交互”的顺序往下学，先建立最基本的前端认知，再进入具体语法和规范。

后面的内容就会按照这两个部分逐页展开，做到一页一个重点，方便大家跟着节奏理解和消化。
